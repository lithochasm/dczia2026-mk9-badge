# MicroPython UF2

`RPI_PICO-20260406-v1.28.0.uf2` is the official stable MicroPython v1.28.0
firmware for the Raspberry Pi Pico/RP2040 target, downloaded from:

<https://micropython.org/download/RPI_PICO/>

SHA-256:

```text
00aedb8431ebea83b470ef909509f16432c5b1b43e8f6eaf015f6f9560bc8152
```

Hold BOOTSEL while plugging in the badge, then copy the UF2 onto the `RPI-RP2`
drive. The drive will disappear when flashing finishes. Upload the Python files
afterward using Thonny or `tools/upload_micropython.sh`.
